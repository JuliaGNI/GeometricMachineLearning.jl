# Homogeneous Spaces 

**Homogeneous spaces** are manifolds $\mathcal{M}$ on which a Lie group $G$ acts transitively, i.e.  

```math
\forall X,Y\in\mathcal{M} \exists{}A\in{}G\text{ s.t. }AX = Y.
```
